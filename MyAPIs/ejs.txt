how to add images/css/javascripts  in express project


a) create "public" folder inside backend folder
b) create "images" folder inside "public"
c) write below line in index.js
    app.use(express.static('public))

ab jaha bhi image add krni hai waha - <img src="images/meet.png" width="80px" />  
note - public folder tk apne aap include rehta hai path mai .  uske aake ka path dena padta hai
